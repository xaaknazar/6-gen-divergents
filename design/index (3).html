# Divergents · Тест 6 Гениев Современный одностраничный сайт для прохождения командного теста «6 Гениев». Человек заходит, вводит имя, проходит 42 вопроса и сразу видит красивый отчёт с ведущими гениями, фрустрациями и radar-графиком. Результаты (по желанию) автоматически уходят в вашу Google Sheets. ## Структура ``` . ├── index.html — разметка (intro, тест, результаты) ├── styles.css — оформление ├── script.js — логика теста, подсчёт, график, отправка ├── questions.js — 42 вопроса с тегами типов (W/I/D/G/E/T) ├── geniuses.js — описания, цвета, иконки 6 типов ├── config.js — конфиг (URL Apps Script) └── apps-script/Code.gs — бекенд для Google Sheets ``` ## Запуск локально Просто откройте `index.html` в браузере. Либо поднимите локальный сервер: ```bash python3 -m http.server 8000 # http://localhost:8000 ``` ## Деплой (бесплатные варианты) - **GitHub Pages**: залить репозиторий, Settings → Pages → source = main. - **Netlify**: перетащить папку на [app.netlify.com/drop](https://app.netlify.com/drop). - **Vercel**: `vercel` в корне проекта. Все файлы — статика, сервер не нужен. ## Сохранение в Google Sheets 1. Откройте Google Sheets, которую хотите использовать (или создайте новую). 2. Меню **Расширения → Apps Script**. Удалите дефолтный код. 3. Скопируйте всё содержимое файла `apps-script/Code.gs` в редактор и сохраните. 4. **Deploy → New deployment**: - Тип: **Web app** - Execute as: **Me** - Who has access: **Anyone** 5. Нажмите **Deploy**, авторизуйте. Скопируйте полученный **Web app URL**. 6. Откройте `config.js` и вставьте URL: ```js window.APP_CONFIG = { SHEETS_WEBHOOK_URL: 'https://script.google.com/macros/s/.../exec' }; ``` 7. Готово. Каждый завершённый тест добавит строку в лист «Результаты» с именем, email, баллами по 6 типам, ведущими гениями и фрустрациями. > Если URL не указан, сайт работает автономно: пользователь видит свой > результат, но ничего не отправляется. ## Логика подсчёта - Каждый вопрос относится к одному из 6 типов: **W** Размышление, **I** Изобретение, **D** Различение, **G** Воодушевление, **E** Поддержка, **T** Упорство. - Варианты ответа: *Не согласен* = 1, *Иногда* = 2, *Полностью согласен* = 3. - По каждому типу 7 вопросов → максимум 21 балл. - Рейтинг: **топ-2 = рабочие гении**, **2 средних = компетенции**, **2 нижних = фрустрации**. ## Кастомизация - **Цвета и тексты типов** → `geniuses.js` - **Список вопросов** → `questions.js` (поле `type` — к какому гению относится) - **Оформление** → `styles.css` (переменные в `:root`) ## Горячие клавиши во время теста - `1` / `2` / `3` — выбор варианта ответа - `←` — вернуться к предыдущему вопросу ## Лицензия Частный проект Divergents. Вопросы и методология основаны на «The 6 Types of Working Genius» (Patrick Lencioni / The Table Group).
